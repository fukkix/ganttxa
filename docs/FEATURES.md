## 项目管理

### 创建项目

**方式一：手动创建**

1. 在首页点击"创建项目"按钮
2. 系统自动创建"未命名项目"
3. 在编辑器中点击项目名称编辑

**方式二：从文件创建**

1. 点击"上传文件"按钮
2. 上传 Excel/Word/CSV 文件
3. AI 自动解析并创建项目

### 编辑项目名称

1. 在编辑器页面，悬停在项目名称上
2. 点击出现的编辑图标 ✏️
3. 输入新名称
4. 按 Enter 保存，或按 Esc 取消
5. 系统自动保存到服务器

### 打开项目

1. 在首页项目列表中
2. 点击项目卡片
3. 进入编辑器页面

### 删除项目

1. 在首页项目列表中
2. 点击项目卡片右下角的删除按钮 🗑️
3. 确认删除操作
4. 项目及所有任务将被永久删除

### 自动保存

- 编辑任务后 30 秒自动保存
- 保存状态显示在项目名称下方
- 显示"保存中..."或"上次保存时间"

## 任务管理

### 创建任务

1. 点击左侧面板的"新建任务"按钮
2. 填写任务信息：
   - **任务名称**（必填）
   - **开始日期**（必填）
   - **结束日期**（可选）
   - **负责人**（可选）
   - **所属阶段**（必填）
   - **任务说明**（可选）
   - **里程碑**（可选）
3. 点击"保存"

### 编辑任务

1. 在任务列表中找到任务
2. 悬停显示操作按钮
3. 点击编辑按钮 ✏️
4. 修改任务信息
5. 点击"保存"

### 删除任务

1. 在任务列表中找到任务
2. 悬停显示操作按钮
3. 点击删除按钮 🗑️
4. 确认删除操作

### 任务排序（拖拽）

1. 展开阶段查看任务列表
2. 悬停在任务上显示拖拽手柄 ⋮⋮
3. 按住鼠标左键拖动任务
4. 拖到目标位置释放
5. 任务顺序自动更新

**拖拽提示**：
- 拖动时任务半透明显示
- 目标位置高亮显示
- 支持跨阶段拖拽
- 自动保存新顺序

### 阶段管理

**展开/折叠阶段**

- 点击阶段标题展开或折叠
- 显示该阶段的任务数量
- 阶段颜色标识

**阶段颜色**

系统预设 5 种阶段颜色：
- 阶段一：蓝色 #3b82f6
- 阶段二：绿色 #10b981
- 阶段三：橙色 #f59e0b
- 阶段四：红色 #ef4444
- 阶段五：紫色 #8b5cf6

### 里程碑

- 创建任务时勾选"里程碑"
- 里程碑任务显示特殊标记
- 在甘特图中以菱形显示

## 甘特图

### 时间粒度

切换不同的时间视图：

- **日视图**：按天显示，适合短期项目
- **周视图**：按周显示，适合中期项目
- **月视图**：按月显示，适合长期项目

### 缩放控制

**放大/缩小**

- 点击 ➕ 按钮放大（最大 300%）
- 点击 ➖ 按钮缩小（最小 50%）
- 点击百分比数字重置为 100%

**缩放范围**：50% - 300%

**快捷键**（即将支持）：
- Ctrl + 滚轮：缩放
- Ctrl + 0：重置缩放

### 今天标记

- 点击"今天"按钮跳转到当前日期
- 甘特图中显示今日线
- 今日线为红色虚线

### 显示选项

**显示周末**

- 切换开关显示/隐藏周末
- 周末以浅灰色背景显示
- 不影响任务计算

**显示节假日**

- 切换开关显示/隐藏节假日
- 节假日以特殊颜色标记
- 支持中国法定节假日

### 任务条

**颜色编码**

- 按阶段自动着色
- 悬停显示任务详情
- 点击跳转到任务编辑

**任务状态**

- 未开始：浅色
- 进行中：正常色
- 已完成：深色
- 逾期：红色

### 依赖关系（即将推出）

- 任务间连线显示依赖
- 支持 4 种依赖类型：
  - FS（完成-开始）
  - SS（开始-开始）
  - FF（完成-完成）
  - SF（开始-完成）

## AI 文件解析

### 支持的文件格式

- **Excel**：.xlsx, .xls
- **Word**：.docx
- **CSV**：.csv

**文件大小限制**：20MB

### 配置 AI

**首次使用**

1. 点击"上传文件"
2. 点击"AI 设置"
3. 选择 API 提供商：
   - **OpenRouter**（推荐）：价格优惠，多模型
   - **Anthropic 官方**：直连，稳定
4. 输入 API Key
5. 选择模型：
   - **Claude Sonnet 4**（推荐）：准确率高
   - **Claude 3.5 Sonnet**：平衡选择
   - **Claude 3 Haiku**：快速便宜
6. 保存设置

**获取 API Key**

- OpenRouter: https://openrouter.ai/keys
- Anthropic: https://console.anthropic.com/settings/keys

详细指南：[OPENROUTER-SETUP.md](OPENROUTER-SETUP.md)

### 上传和解析

1. 点击"上传文件"按钮
2. 拖拽或选择文件
3. 系统自动上传
4. AI 开始解析（10-30秒）
5. 显示解析进度

### 字段映射

解析完成后：

1. 查看 AI 识别的字段映射
2. 确认或调整映射关系：
   - 任务名称
   - 开始日期
   - 结束日期
   - 负责人
   - 所属阶段
   - 任务说明
3. 实时预览解析结果
4. 点击"确认"导入任务

### 准确率

- 目标准确率：≥90%
- 置信度评分：0-1
- 低置信度字段高亮提示
- 支持手动调整

### 文件格式建议

**Excel/CSV**

- 使用清晰的列标题
- 推荐标题：
  - 任务名称、工作项、活动
  - 开始时间、起始日期
  - 结束时间、完成日期
  - 负责人、执行人
  - 阶段、模块
- 日期格式：YYYY-MM-DD 或 YYYY年MM月DD日
- 避免合并单元格

**Word**

- 使用表格组织任务
- 清晰的标题行
- 避免复杂的嵌套结构

## 分享协作

### 生成分享链接

1. 点击右上角"分享"按钮
2. 设置权限级别：
   - **仅查看**：只能查看，不能编辑
   - **可评论**：可以添加评论
   - **可编辑**：可以修改任务
3. 设置过期时间（可选）：
   - 1 天
   - 7 天
   - 30 天
   - 永久
4. 点击"生成链接"
5. 复制链接发送给团队成员

### 分享链接特性

- **无需注册**：访问者无需创建账号
- **加密链接**：24+ 字符随机字符串
- **权限控制**：三级权限管理
- **过期管理**：自动过期或手动撤销
- **访问统计**：查看访问次数（可选）

### 撤销分享

1. 打开分享对话框
2. 点击"撤销链接"
3. 确认操作
4. 链接立即失效

### 更新权限

1. 打开分享对话框
2. 修改权限级别或过期时间
3. 点击"更新"
4. 新设置立即生效

## 导出功能

### PNG 导出

**用途**：演示、报告、打印

**特性**：
- 2x 高清分辨率
- 完整甘特图视图
- 包含时间轴和任务条
- 透明背景（可选）

**步骤**：
1. 点击"导出"按钮
2. 选择"PNG 图片"
3. 点击"下载"
4. 保存到本地

### CSV 导出

**用途**：Excel 编辑、数据分析

**包含字段**：
- 任务名称
- 开始日期
- 结束日期
- 负责人
- 所属阶段
- 任务说明
- 进度
- 状态

**步骤**：
1. 点击"导出"按钮
2. 选择"CSV 表格"
3. 点击"下载"
4. 用 Excel 打开

### JSON 导出

**用途**：数据备份、API 集成

**包含内容**：
- 完整项目信息
- 所有任务数据
- 甘特图配置
- 元数据

**步骤**：
1. 点击"导出"按钮
2. 选择"JSON 数据"
3. 点击"下载"
4. 保存备份文件

### PDF 导出（即将推出）

**用途**：正式文档、归档

**特性**：
- A4/A3 横版布局
- 300dpi 高清晰度
- 自动分页
- 页眉页脚

## 常见问题

### Q: 如何批量导入任务？

A: 使用 AI 文件解析功能，上传 Excel/Word/CSV 文件即可批量导入。

### Q: 任务可以跨阶段拖拽吗？

A: 可以，拖拽任务到其他阶段即可自动更新阶段归属。

### Q: 分享链接可以设置密码吗？

A: 当前版本暂不支持，计划在 M3 阶段添加。

### Q: 支持甘特图打印吗？

A: 支持，导出为 PNG 或 PDF（即将推出）后打印。

### Q: 如何备份项目数据？

A: 导出为 JSON 格式即可完整备份所有数据。

### Q: AI 解析费用如何计算？

A: 使用您自己的 API Key，费用由 AI 提供商收取。典型项目约 $0.002-0.03/次。
